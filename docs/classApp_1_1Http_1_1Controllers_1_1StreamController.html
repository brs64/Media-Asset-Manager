App\Http\Controllers\StreamController::stream ( Request  $request,
  $mediaId 
)

Point d'entrée principal pour le streaming d’un média.

Cette méthode détermine automatiquement la source la plus appropriée pour servir la vidéo en fonction du contexte et des données disponibles.

Ordre de priorité :

  1. Environnement local → fichiers locaux (dev)
  2. Fichier transcodé local (chemin_local)
  3. NAS via FTP (ARCH ou PAD)

Supporte :

  • Streaming MP4 avec gestion des requêtes HTTP Range (seek vidéo)
  • Streaming HLS (.m3u8)

Inclut un fallback en environnement de développement si la base de données est inaccessible ou si le média n’est pas trouvé.

Paramètres
Request$requestRequête HTTP entrante
int | string$mediaIdIdentifiant du média à streamer
Renvoie
\Symfony\Component\HttpFoundation\StreamedResponse|\Illuminate\Http\Response
Exceptions

Symfony\Component\HttpKernel\Exception\HttpException Si le média ou la source vidéo est introuvable

private

Tente de résoudre et streamer une vidéo locale à partir d’un modèle Media.

Cette méthode applique plusieurs stratégies pour retrouver un fichier local :

  • Nom technique exact du média
  • Nom sans extension avec ajout ".mp4"
  • Nom avec espaces remplacés par des underscores
  • Nom extrait depuis les URI disponibles

En cas d’échec, un fallback est appliqué en sélectionnant le premier fichier MP4 disponible dans le dossier cible.

Stream un fichier vidéo MP4 depuis un stockage distant avec support HTTP Range.

Permet une lecture fluide et optimisée en supportant :

  • les requêtes partielles (seek vidéo)
  • le streaming progressif

Le fichier est lu en flux (stream) depuis un disque Laravel (FTP), avec gestion manuelle des offsets et des buffers.

string$ftpDiskNom du disque Laravel configuré (ex: ftp_arch, ftp_pad)
string$videoPathChemin du fichier vidéo sur le stockage distant
Request$requestRequête HTTP (utilisée pour détecter les headers Range)
Renvoie
StreamedResponse
Exceptions

Symfony\Component\HttpKernel\Exception\HttpException Si le fichier est introuvable ou en cas d'erreur de streaming
